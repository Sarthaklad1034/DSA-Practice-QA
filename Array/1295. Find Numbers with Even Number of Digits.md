## ğŸ› ï¸ Solution Approaches

### âœ… Approach 1: String Conversion Check (Java)

For each number, convert it to a string and check if its length is even.

```java
class Solution {
    public int findNumbers(int[] nums) {
        int evenCount = 0;

        for (int num : nums) {
            if (String.valueOf(num).length() % 2 == 0) {
                evenCount++;
            }
        }

        return evenCount;
    }
}
```

**Time Complexity:** `O(n * d)`  
**Space Complexity:** `O(1)`  
Where `n` is the number of elements and `d` is the average number of digits.

### ğŸš€ Approach 2: Optimized Math Digit Count (Java)

Avoids converting to string. Uses math to count digits and check evenness.

```java
class Solution {
    public int findNumbers(int[] nums) {
        int evenCount = 0;

        for (int num : nums) {
            int digits = 0;
            while (num > 0) {
                digits++;
                num /= 10;
            }
            if (digits % 2 == 0) {
                evenCount++;
            }
        }

        return evenCount;
    }
}
```

**Space Complexity:** `O(1)`  
Where `n` is the array length and `m` is the maximum value.

### ğŸ Approach 3: Python - String Length Check

Pythonic way using string conversion and loop.

```python
class Solution:
    def findNumbers(self, nums: List[int]) -> int:
        evenCount = 0

        for num in nums:
            if len(str(num)) % 2 == 0:
                evenCount += 1

        return evenCount
```

**Time Complexity:** `O(n * d)`  
**Space Complexity:** `O(1)`
